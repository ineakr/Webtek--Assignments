Questions

  1. The server recives requests from the client, and sends services back to the client. For example can the server connect the client to a webpage.
  2. The client is where the user operate. The client requests services from the server and then recives it. Then, the user can for example display a website.
  3. The IP stands for Internet Protocol, and is a set of rules for the internet. It makes it possible to communicate over the internet. The IP address is an unique code for a website or a device connected to the internet.
  4. DNS stands for Domain Name System. When an user wants to visit a webpage, the DNS recives a domain name and returns the IP address for the website. If the DNS don't have the IP address, it sends it to another DNS until it recives the right address. The DNS saves the IP address for the next time the user wants to visit the page.
  5. The difference between the Internet and the World Wide Web is that the internet is a network of networks. The internet is a connection between many networks. The world wide web on the otherhand, is an application on the internet. For example a website.
  6. HTML is the code language for the structure of a website.
